Colors

ID vs Class:
ID: Unique, one per page.
Class: Can be used multiple times.

SPAN vs DIV:
SPAN: Inline element (small content).
DIV: Block-level element (larger sections).

DIV vs HEADER, MAIN, FOOTER:
IV: Generic container.
HEADER: Intro content.
MAIN: Primary content.
FOOTER: Bottom content.

Lorem Color Selection:
 Named (e.g., red), Hex (e.g., #ff5733), RGB (e.g., rgb(255, 0, 0)), RGBA, HSL, HSLA, CSS Variables.
